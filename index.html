HI, MY NAME IS Efehan

Hallo, ik ben een software-student en ik wil me graag aan jullie voorstellen. Mijn naam is Faruk Efehan YILDIZ en ik studeer software engineering. Ik kom uit Turkije, ik ben pas 1,5 jaar in Nederland. Op dit moment studeer ik voorbereidend jaar aan de Universiteit Leiden en ontwikkel ik mijn Nederlands. Ik was de beste school van mijn stad in Turkije en kan zeggen dat het gelijk is aan VWO in het Nederlandse onderwijssysteem. Ik wil graag carrière maken in de software engineering en daarom leer ik voortdurend nieuwe technologieën. Ik vind het leuk om code te schrijven en ik ben enthousiast over het toepassen van wat ik leer. Ik hou niet alleen van software engineering, maar ik ben ook dol op sporten. Ik ga regelmatig naar de sportschool en speel basketbal. Bovendien heb ik mijn ervaring in software engineering ontwikkeld door deel te nemen aan een bootcamp
